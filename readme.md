## Install

```shell
npm install -g jsherlock-cli
```

[![asciicast](https://asciinema.org/a/223172.svg)](https://asciinema.org/a/223172)

## Usage

```shell
Usage
  $ jsherlock <input>

Options
  --export-csv='jsherlock.csv'

Examples
  $ jsherlock zuck
  $ jsherlock zuck --export-csv='jsherlock.csv'
```
